# 历史性能档案

本目录只保存**未在本次文档修复中重新复现**的历史性能分析、历史 benchmark 快照与调优背景材料。

## 使用边界

- 当前可验证的 benchmark target、源码路径与运行命令，以 `docs/05-性能测试.md` 为准
- 本目录中的数字、排名、结论默认都属于**历史数据**，不能直接当作当前版本性能承诺
- 如果需要在 README、主导航或对外交付材料中引用吞吐/延迟数据，必须先重新执行对应 benchmark
