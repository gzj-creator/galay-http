# 03-使用指南

本文档提供 Galay-HTTP 的详细使用说明、最佳实践和常见场景。

## 选择合适的组件

### HTTP vs WebSocket vs HTTP/2

| 特性 | HTTP/1.1 | WebSocket | HTTP/2 |
|------|----------|-----------|--------|
| 适用场景 | RESTful API、Web 服务 | 实时通信、推送 | 高性能 API、多路复用 |
| 连接模式 | 请求-响应 | 全双工 | 多路复用 |
| 性能 | 中等 | 高（长连接） | 高（多路复用） |
| 复杂度 | 简单 | 中等 | 较高 |

### 使用建议

**选择 HTTP/1.1 当**:
- 构建标准 RESTful API
- 需要广泛的客户端兼容性
- 简单的请求-响应模式
- 静态文件服务

**选择 WebSocket 当**:
- 需要实时双向通信
- 服务器主动推送数据
- 聊天、游戏、实时监控
- 减少轮询开销

**选择 HTTP/2 当**:
- 需要高性能和低延迟
- 多个并发请求
- 头部压缩优化
- 服务器推送

## HTTP 服务器开发指南

### 基础服务器设置

```cpp
#include "galay-http/kernel/http/HttpServer.h"
#include "galay-http/kernel/http/HttpRouter.h"
#include "galay-http/utils/Http1_1ResponseBuilder.h"

using namespace galay::http;
using namespace galay::kernel;

int main() {
    // 配置服务器
    HttpServerConfig config;
    config.host = "0.0.0.0";
    config.port = 8080;
    config.backlog = 128;
    config.io_scheduler_count = 0;       // 0 = 自动（CPU 核心数）
    config.compute_scheduler_count = 0;  // 0 = 自动

    // 创建路由器
    HttpRouter router;

    // 添加路由处理器
    router.addHandler<HttpMethod::GET>("/", [](HttpConn& conn, HttpRequest req) -> Coroutine {
        auto response = Http1_1ResponseBuilder::ok()
            .text("Hello, World!")
            .build();

        auto writer = conn.getWriter();
        while (true) {
            auto result = co_await writer.sendResponse(response);
            if (!result || result.value()) break;
        }
        co_return;
    });

    // 启动服务器
    HttpServer server(config);
    server.start(std::move(router));

    std::cout << "Server running on http://0.0.0.0:8080\n";

    // 保持运行
    while (true) {
        std::this_thread::sleep_for(std::chrono::seconds(1));
    }

    return 0;
}
```

```cpp

    auto& header = req.header();

    // 获取 HTTP 方法
    HttpMethod method = header.method();

    // 获取 URI
    std::string uri = header.uri();

    // 获取 HTTP 版本
    HttpVersion version = header.version();

    }

}
```

## HTTP 客户端开发指南

### 基本请求

```cpp
#include "galay-http/kernel/http/HttpClient.h"
#include "galay-http/utils/Http1_1RequestBuilder.h"
#include "galay-kernel/kernel/Runtime.h"

using namespace galay::http;
using namespace galay::kernel;

Coroutine sendRequest() {
    HttpClient client;

    // 连接服务器
    auto connect_result = co_await client.connect("http://example.com:80");
    if (!connect_result) {
        std::cerr << "连接失败\n";
        co_return;
    }

    // 获取 Session
    auto session = client.getSession();
    auto writer = session.getWriter();
    auto reader = session.getReader();

    // 构建请求
    auto request = Http1_1RequestBuilder::get("/api/data")
        .host("example.com")
        .header("User-Agent", "Galay-HTTP/1.0")
        .build();

    // 发送请求
    while (true) {
        auto send_result = co_await writer.sendRequest(request);
        if (!send_result || send_result.value()) break;
    }

    // 接收响应
    HttpResponse response;
    auto recv_result = co_await reader.getResponse(response);
    if (!recv_result) {
        std::cerr << "接收响应失败\n";
        auto close_result = co_await client.close();
        if (!close_result) {
            std::cerr << "关闭连接失败\n";
        }
        co_return;
    }

    std::cout << "状态码: " << response.getStatus() << "\n";
    std::cout << "响应体: " << response.getBodyStr() << "\n";

    auto close_result = co_await client.close();
    if (!close_result) {
        std::cerr << "关闭连接失败\n";
    }
}
```

## WebSocket 开发指南

### WebSocket 服务器

```cpp
#include "galay-http/kernel/websocket/WsUpgrade.h"
#include "galay-http/kernel/websocket/WsConn.h"

using namespace galay::websocket;

Coroutine handleWebSocket(WsConn& ws_conn) {
    auto reader = ws_conn.getReader(WsReaderSetting{
        .max_frame_size = 1024 * 1024,        // 1MB
        .max_message_size = 10 * 1024 * 1024, // 10MB
        .auto_handle_ping = true,
        .auto_handle_close = true
    });

    auto writer = ws_conn.getWriter(WsWriterSetting::byServer());

    // 发送欢迎消息
    auto welcome_result = co_await writer.sendText("Welcome!");
    if (!welcome_result) {
        co_return;
    }

    // 消息循环
    while (true) {
        std::string message;
        WsOpcode opcode;

        auto result = co_await reader.getMessage(message, opcode);

        if (!result.has_value()) {
            // 错误处理
            if (result.error().code() == WsError::Code::CONNECTION_CLOSED) {
                std::cout << "连接已关闭\n";
            }
            break;
        }

        if (!result.value()) {
            // 未完成，继续读取
            continue;
        }

        // 处理消息
        if (opcode == WsOpcode::Text) {
            std::cout << "收到文本: " << message << "\n";

            // 回显消息
            auto echo_result = co_await writer.sendText("Echo: " + message);
            if (!echo_result) {
                break;
            }
        } else if (opcode == WsOpcode::Binary) {
            std::cout << "收到二进制数据: " << message.size() << " 字节\n";
        } else if (opcode == WsOpcode::Close) {
            std::cout << "收到关闭帧\n";
            break;
        }
    }

    auto close_result = co_await ws_conn.close();
    if (!close_result) {
        std::cerr << "关闭 WebSocket 连接失败\n";
    }
}

Coroutine wsHandler(HttpConn& conn, HttpRequest req) {
    WsUpgrade upgrade;
    auto ws_conn = co_await upgrade.upgrade(conn, req);

    if (ws_conn) {
        co_await handleWebSocket(*ws_conn);
    } else {
        std::cerr << "WebSocket 升级失败\n";
    }

    co_return;
}

int main() {
    HttpRouter router;
    router.addHandler<HttpMethod::GET>("/ws", wsHandler);

    HttpServerConfig config;
    config.port = 8080;

    HttpServer server(config);
    server.start(std::move(router));

    std::cout << "WebSocket server running on ws://0.0.0.0:8080/ws\n";

    while (true) {
        std::this_thread::sleep_for(std::chrono::seconds(1));
    }

    return 0;
}
```

### WebSocket 客户端

```cpp
#include "galay-http/kernel/websocket/WsClient.h"

Coroutine wsClientExample() {
    WsClient client;

    // 连接服务器
    auto connect_result = co_await client.connect("ws://127.0.0.1:8080/ws");
    if (!connect_result) {
        std::cerr << "连接失败\n";
        co_return;
    }

    // 获取 Session 并升级
    auto session = client.getSession(WsWriterSetting::byClient());
    auto upgrader = session.upgrade();
    auto upgrade_result = co_await upgrader();
    if (!upgrade_result) {
        std::cerr << "升级失败\n";
        co_return;
    }

    // 获取连接
    auto conn = session.getConn();
    if (!conn) {
        std::cerr << "获取连接失败\n";
        co_return;
    }

    auto reader = conn->getReader(WsReaderSetting{});
    auto writer = conn->getWriter(WsWriterSetting::byClient());

    // 发送消息
    auto send_result = co_await writer.sendText("Hello, WebSocket!");
    if (!send_result) {
        std::cerr << "发送失败\n";
        co_return;
    }

    // 接收消息
    std::string message;
    WsOpcode opcode;
    auto result = co_await reader.getMessage(message, opcode);
    if (result.has_value() && result.value()) {
        std::cout << "收到: " << message << "\n";
    }

    // 发送 Ping
    auto ping_result = co_await writer.sendPing("ping");
    if (ping_result) {
        std::cout << "Ping 发送成功\n";
    }

    // 关闭连接
    auto close_result = co_await writer.sendClose(1000, "Normal closure");
    if (close_result) {
        std::cout << "关闭帧发送成功\n";
    }

    auto close_conn_result = co_await conn->close();
    if (!close_conn_result) {
        std::cerr << "关闭连接失败\n";
    }
}
```

## 错误处理最佳实践

### HTTP 错误处理

```cpp
Coroutine handleWithErrorCheck(HttpConn& conn, HttpRequest req) {
    try {
        // 处理请求...
        auto response = Http1_1ResponseBuilder::ok()
            .text("Success")
            .build();

        auto writer = conn.getWriter();
        while (true) {
            auto result = co_await writer.sendResponse(response);
            if (!result) {
                // 发送失败

                break;
            }
            if (result.value()) {
                // 发送完成
                break;
            }
        }

        std::cerr << "异常: " << e.what() << '\n';

        auto error_response = Http1_1ResponseBuilder::internalServerError()
            .json("{\"error\":\"Internal server error\"}")
            .build();

        auto writer = conn.getWriter();
        while (true) {
            auto result = co_await writer.sendResponse(error_response);
            if (!result || result.value()) break;
        }
    }

    co_return;
}
```

```cpp
if (!close_result) {
    co_return;
}
```

## 常见问题

### Q: 如何处理大文件上传？

A: 使用分块传输编码（Chunked Transfer Encoding）或设置合适的缓冲区大小。

### Q: WebSocket 如何实现心跳？

A: 定期发送 Ping 帧：

```cpp
while (true) {
    co_await galay::kernel::sleep(std::chrono::seconds(30));
    auto ping_result = co_await writer.sendPing("heartbeat");
    if (!ping_result) {
        break;
    }
}
```

### Q: 如何实现 CORS？

A: 在响应中添加 CORS 头部：

```cpp
auto response = Http1_1ResponseBuilder::ok()
    .header("Access-Control-Allow-Origin", "*")
    .header("Access-Control-Allow-Methods", "GET, POST, PUT, DELETE")
    .header("Access-Control-Allow-Headers", "Content-Type")
    .json("{\"data\":\"value\"}")
    .build();
```

### Q: 如何实现请求超时？

A: 使用 galay-kernel 的超时机制：

```cpp
auto timeout_result = co_await galay::kernel::timeout(
    std::chrono::seconds(5),
    [&]() -> Coroutine {
        auto send_result = co_await writer.sendRequest(request);
        if (!send_result || !send_result.value()) {
            co_return;
        }
        HttpResponse response;
        auto recv_result = co_await reader.getResponse(response);
        if (!recv_result || !recv_result.value()) {
            co_return;
        }
        co_return;
    }()
);
```
