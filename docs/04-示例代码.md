# 04-示例代码

本页只列出 `examples/CMakeLists.txt` 中真实存在的 target，并为每个示例给出真实源码路径、运行命令、依赖条件与关联测试。本页是唯一示例索引入口，目录级 `examples/README.md` 已不再保留。

## 构建前提

### 明文示例

```bash
cmake -S . -B build \
  -DCMAKE_BUILD_TYPE=Release \
  -DBUILD_EXAMPLES=ON \
  -DBUILD_MODULE_EXAMPLES=OFF
cmake --build build --parallel
```

### TLS 示例

```bash
cmake -S . -B build-ssl \
  -DCMAKE_BUILD_TYPE=Release \
  -DGALAY_HTTP_ENABLE_SSL=ON \
  -DBUILD_EXAMPLES=ON \
  -DBUILD_MODULE_EXAMPLES=OFF
cmake --build build-ssl --parallel
```

说明：

- TLS 示例需要构建开关 `-DGALAY_HTTP_ENABLE_SSL=ON`
- 运行时无需额外环境变量；若使用自签名证书，可直接复用 `test/test.crt` 与 `test/test.key`

## 重要说明

### 文件名与路径以真实仓库命名为准

示例源码全部使用 `lower_snake_case` 文件名，例如：

- `examples/include/E1-echo_server.cpp`
- `examples/include/E5-https_server.cpp`
- `examples/include/E9-h2c_echo_server.cpp`
- `examples/include/E13-h2_echo_server.cpp`

不要再使用旧文档里的 `E1-EchoServer.cpp`、`E5-HttpsServer.cpp` 之类路径。

### WSS 服务端的能力边界

`E7-WssServer` 是真实可构建示例，但它不是独立高层 `WssServer` API 的证明。文件头已说明当前实现通过 `HttpsServer` + 手动帧处理工作，这一点必须与 `README.md` 和 `docs/03-使用指南.md` 一起理解。
